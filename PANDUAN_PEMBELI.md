# 🚀 Cara Install NetDoc AI OS (± 3 Menit)

Terima kasih sudah membeli **NetDoc AI OS™**! Ikuti 3 langkah ini untuk
mulai pakai aplikasinya di HP Anda sendiri.

---

## Langkah 1 — Deploy Aplikasi (Gratis, Punya Anda Sendiri)

1. Klik tombol/link berikut yang diberikan penjual:
   **[Deploy Sekarang →](#)** *(link dari penjual)*
2. Anda akan diarahkan ke situs **Vercel** — kalau belum punya akun, klik
   **"Continue with GitHub"** untuk daftar (gratis, ±1 menit, cukup pakai
   email).
3. Vercel akan menampilkan halaman "Create New Project" — biarkan semua
   pengaturan default, klik tombol biru **"Deploy"**.
4. Tunggu ±1 menit sampai muncul tulisan **"Congratulations!"** dengan
   tampilan aplikasi Anda.

Sekarang aplikasi ini **milik Anda sepenuhnya** — berjalan di akun Anda
sendiri, gratis selamanya, tidak tergantung pada penjual.

---

## Langkah 2 — Buka Link-nya

1. Di halaman "Congratulations", klik tombol **"Continue to Dashboard"**
   atau klik gambar preview situsnya.
2. Salin URL yang muncul, formatnya seperti:
   `https://netdoc-ai-os-xxxxx.vercel.app`
3. Simpan link ini — inilah alamat aplikasi Anda.

---

## Langkah 3 — Install ke Layar Utama HP

1. Buka link dari Langkah 2 **di HP Anda**:
   - **Android** → buka pakai **Chrome**
   - **iPhone** → buka pakai **Safari**
2. **Android**: tunggu beberapa detik, akan muncul pop-up "Install" di
   bawah layar → tap **Install**.
   (Kalau tidak muncul: tap menu ⋮ pojok kanan atas → **"Install app"**)
3. **iPhone**: tap ikon **Share** (kotak panah ke atas) → scroll → tap
   **"Add to Home Screen"** → tap **"Add"**.
4. Ikon aplikasi muncul di layar utama HP Anda — selesai! 🎉

---

## ❓ Pertanyaan Umum

**Apakah data saya aman?**
Semua data (Asset Database, hasil audit, dll) tersimpan **hanya di HP/
browser Anda sendiri** (localStorage). Tidak dikirim ke server penjual
atau siapa pun. Backup berkala lewat menu **Lainnya → Export Backup**.

**Apakah perlu bayar hosting?**
Tidak. Vercel gratis untuk penggunaan pribadi seperti ini selamanya.

**Bisa dipakai di banyak HP?**
Bisa — buka link yang sama di HP lain, lalu install dengan cara yang sama.
Catatan: karena data tersimpan lokal per-HP, data di HP A dan HP B akan
terpisah kecuali Anda export/import backup secara manual.

**Saya tidak paham teknis, apa nanti bisa minta bantuan?**
Hubungi penjual sesuai kanal yang tertera di halaman pembelian Anda.
